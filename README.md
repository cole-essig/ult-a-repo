# Ult-a-repo

A React-based personal learning project that integrates multiple apps using React Router DOM. This project is designed to enhance programming skills in JavaScript, React, and Python, with REST APIs being built in conjunction with it. The backend is TBD and will be linked later.

## Features

- **Password Generator** – Secure password creation
- **Notepad & Idea Tracker** – Save and organize thoughts
- **Checklist & To-Do List** – Keep track of tasks
- **Calculator** – Perform basic calculations
- **Tic-Tac-Toe** – Classic two-player game
- **Connect 4** – Strategy-based board game
- **Local Weather App** – Displays current weather
- **Currency Exchange** – Converts currency rates
- **Photo Uploader** – Upload and manage images
- **Fun Fact Quiz** – AI-powered multiple-choice quiz
- **Weekly Calendar** – View holidays and special days worldwide

## Technologies

- **Frontend:** React, React Router DOM, Webpack
- **Backend:** Python REST APIs (TBD)

## Installation

### Prerequisites

Ensure you have the following installed:

- **Node.js** (latest LTS)
- **npm** or **yarn**
- **Python** (for backend APIs)

### Clone the Repository

git clone https://github.com/yourusername/ult-a-repo.git
cd ult-a-repo

### Install Dependencies

npm install

### Running App

npm run dev
